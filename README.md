
# FPS_Mines

## Общая концепция

3D-интерпретация классической игры «Сапёр». Игрок исследует минное поле от первого лица как человек-сапёр. Цель – разминировать n-размерное минное поле, для проверки можно будет открывать меню Tab, которое будет выглядеть аналогично классическому «Сапёру».

## Инфа по проекту
жанр: 3D головоломка / симулятор от  первого лица

платформа: PC(Windows)
